## Run Locally
Make sure to have Flutter installed on your computer

Clone the project

```bash
  git clone https://github.com/Souradeep1101/Calculator
```

Go to the project directory

```bash
  cd <path/to/Calculator>
```

Install dependencies

```bash
  flutter pub get
```

Run the App

```bash
  flutter run
```
